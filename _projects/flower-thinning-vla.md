This project explores robotic learning for flower thinning, a manipulation task that requires target selection, precise contact, and reliable execution in cluttered agricultural scenes.

The current implementation uses imitation-learning policies and visual target priors to move from teleoperated demonstrations toward closed-loop real-robot deployment. The project is also a foundation for testing VLA models and reward/value learning in long-horizon manipulation.

MaskACT is one concrete outcome of this direction: an ACT-based behavior cloning policy augmented with a binary visual mask prior for robotic flower thinning.
