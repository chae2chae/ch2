This project integrates navigation, perception, vertical lift motion, and manipulation for agricultural work inside narrow indoor farm aisles.

The system direction is broader than a single end-effector: it frames pollination and harvesting as repeatable robot workflows that combine field perception, task sequencing, and closed-loop actuation.
