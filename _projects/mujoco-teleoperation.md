This project built a MuJoCo-based teleoperation interface that bridges hand-tracking, Vive Tracker, and Manus glove signals to simulated robot arms and dexterous hand grippers.

The work is relevant to imitation learning because high-quality teleoperation can provide demonstrations, expose failure cases, and create a bridge from human motion to robot policy learning.
